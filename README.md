# Fit & Meets-結合穿戴式裝置之適地性交友APP
在畢業專題發想時，發現身邊許多單身朋友都對交友軟體有興趣，並且我們在現行的交友APP上發現兩個缺點：

1. 雖然能搜尋附近朋友，但也還是侷限在虛擬上
2. 使用者黏度低，當要到聯絡方式後就易脫離平台

所以我們決定畢業專題想做一款校園交友APP，試著解決以上兩個缺點，

- 利用打破虛實的功能使交友突破虛擬
- 適地性的交友任務增加使用者黏度

期望校園中不同學院的學生們能走出自己的學院，認識更多異性，或是激發更多運動動機。

###Demo影片：https://youtu.be/h5iO0Kkj8YY


##Fit & Meets是一款交友運動APP：
- 幫助你養成運動習慣
- 將心率分析融入進交友中
- Location Based Service
- Avatar概念設計讓你身歷其境
- 可以認識校園中不同學院的異性朋友

##Location Based Service

###何謂 LBS ?

利用定位技術確定行動通訊設備的位置，並提供各種與位置相關應用的加值服務。

又稱為適地性服務(Location Based Services) 著名應用: Pokemon Go

###Fit & Meets：
- 使得虛擬遊戲不再只侷限在虛擬中
- 虛實整合的遊戲能增加大家外出運動的誘因
